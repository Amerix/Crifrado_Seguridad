Proyecto crifrado 

## Descripción
Aplicación web para cifrar y descifrar texto usando métodos clásicos.

## Características
- Cifrado César configurable
- Cifrado Atbash
- Soporte ASCII
- Interfaz web

## Seguridad
- Validación de entradas
- Funciones puras
- Sin uso de eval()
- Código documentado

## Uso
1. Seleccionar método
2. Introducir texto
3. Procesar
